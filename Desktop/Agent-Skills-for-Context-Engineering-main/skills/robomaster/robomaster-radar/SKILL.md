# RoboMaster 雷达系统

## 系统架构

雷达系统采用四层架构设计：

### 数据采集层
- **相机**：广角相机、全景相机
- **激光雷达**：辅助测距
- **红外传感器**：低光照辅助
- **裁判系统数据**：敌方机器人位置

### 目标检测层
- **目标识别**：敌方机器人检测
- **装甲板检测**：精准定位
- **类型分类**：步兵、英雄、工程等
- **置信度计算**：输出检测置信度

### 目标跟踪层
- **多目标跟踪（MOT）**：同时跟踪多个目标
- **轨迹预测**：卡尔曼滤波、粒子滤波
- **身份关联**：SORT/DeepSORT 算法

### 态势输出层
- **敌方位置**：实时坐标输出
- **威胁等级**：评估优先级
- **运动轨迹**：历史路径显示
- **预警信息**：危险目标提示

## 目标检测方案

### 深度学习方案
- **YOLOv5/YOLOv8**：实时目标检测
- **PyTorch 框架**：模型训练与推理
- **优势**：鲁棒性强，泛化能力好

### 传统视觉方案（OpenCV）
- **图像预处理**：去噪、增强
- **特征提取**：HOG、SIFT、颜色特征
- **目标检测**：模板匹配、轮廓分析
- **优势**：速度快，资源消耗低

## 多目标跟踪

### SORT 算法
- **核心思想**：卡尔曼滤波预测 + 匈牙利算法关联
- **步骤**：
  1. 状态预测
  2. 检测关联
  3. 状态更新
  4. 轨迹管理

### DeepSORT 算法
- **改进点**：加入外观特征（ReID）进行关联
- **优势**：提升 ID 切换鲁棒性

## 威胁评估

### 评估维度
- **距离**：距离越近威胁越高
- **速度**：移动速度越快威胁越高
- **方向**：朝向己方威胁更高
- **类型**：英雄 > 步兵 > 其他

### 威胁等级划分
- **高威胁**：近距离、高速接近
- **中威胁**：中距离、移动中
- **低威胁**：远距离、静止

## 传感器融合

融合多种数据源提升可靠性：
- **视觉数据**：精准定位
- **裁判系统数据**：绝对位置
- **激光雷达数据**：距离测量
