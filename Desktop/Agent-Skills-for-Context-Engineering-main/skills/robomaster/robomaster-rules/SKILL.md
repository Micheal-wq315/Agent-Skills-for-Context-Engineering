# RoboMaster 裁判系统规则讲解

## 比赛概述

RoboMaster 机甲大师超级对抗赛是由大疆创新（DJI）主办的大学生机器人竞技比赛。2026 赛季的比赛形式为红蓝双方对抗，每支队伍需要自主研发多种功能的机器人，在规定场地内进行激烈的战术对抗。

比赛采用类似王者荣耀的对抗模式，要求参赛队独立设计制作多个不同兵种的机器人，参与机器人对抗赛。各机器人通过发射弹丸与敌方机器人及基地进行交互。

## 机器人类型详解

### 步兵机器人
- **数量**：每队 2-3 台
- **定位**：灵活机动的输出单位
- **武器**：17mm 标准弹丸
- **核心技术**：麦克纳姆轮/全向轮底盘，小陀螺战术
- **电控架构**：双板控制（底盘+云台）

### 英雄机器人
- **定位**：主力输出火力平台
- **武器**：42mm 大弹丸
- **特点**：单发伤害高，攻击力强
- **血量**：较高

### 工程机器人
- **定位**：辅助支援单位
- **功能**：兑换金币、救援阵亡友军、触发场地机关
- **特点**：不配备攻击能力

### 空中机器人
- **定位**：空中支援视角
- **平台**：四旋翼飞行平台
- **功能**：空中观测、战术信息传递

### 哨兵机器人
- **定位**：基地自动防御单位
- **特点**：完全自主运行，无需手动操控
- **移动范围**：基地轨道
- **功能**：自动识别并攻击敌方机器人

### 雷达机器人
- **定位**：战场态势感知
- **部署**：己方半场
- **功能**：敌方机器人位置和运动状态探测

## 比赛流程与胜负判定

### 比赛时间结构
- 标准时长：7 分钟
- 阶段划分：准备阶段 → 启动阶段 → 战术阶段 → 基地摧毁阶段

### 胜负判定机制
- **核心依据**：基地剩余血量
- **获胜条件**：比赛结束时基地剩余血量高的一方获胜
- **提前获胜**：一方基地血量提前归零，另一方立即获胜

### 血量系统
每个机器人拥有独立的血量值，被弹丸攻击时根据弹丸类型和命中部位扣除相应血量。血量归零时机器人进入"死亡"状态。

## 场地机关与增益系统

### 地形要素
比赛场地包含多种地形元素：
- 标准地面
- 飞坡
- 资源岛
- 高地
- 基地

### 机关类型
场地机关包括四种类型：
1. **增益机关**：提供作战增益
2. **能量机关**：激活特殊能力
3. **兑换机关**：金币兑换
4. **救援点**：救援阵亡友军

## 操作员界面系统

操作员界面是选手控制机器人的核心终端系统，提供全面的战场信息和控制功能。
