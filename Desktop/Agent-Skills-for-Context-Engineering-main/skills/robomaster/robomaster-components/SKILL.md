# RoboMaster 硬件选型指南

## 电控常用电机

### M3508 减速直流电机
- **应用场景**：底盘驱动
- **额定电压**：DC 24V
- **峰值扭矩**：3.7 N·m
- **额定扭矩**：2.5 N·m
- **空载转速**：480 rpm
- **减速比**：19:1
- **配套电调**：C620

### M2006 P36 直流无刷减速电机
- **应用场景**：精密控制、辅助机构
- **额定电压**：DC 24V
- **峰值扭矩**：0.75 N·m
- **额定扭矩**：0.5 N·m
- **减速比**：36:1
- **配套电调**：C610

### GM6020 高速电机
- **应用场景**：云台控制
- **额定电压**：DC 24V
- **空载转速**：320 rpm
- **额定扭矩**：1.2 N·m
- **额定电流**：1.62A
- **内置电调**：无需外部驱动

## 开发板选型

### RoboMaster C 型开发板
- **主控芯片**：STM32F4 系列
- **特点**：内置 IMU、双 CAN 总线
- **应用**：云台控制、传感器融合
- **优势**：官方支持，资料齐全

### RoboMaster A 型开发板
- **主控芯片**：STM32F4 系列
- **特点**：丰富 PWM 输出
- **应用**：底盘控制
- **优势**：专为电机驱动优化

### 自研板方案
- **参考设计**：STM32F407/F427/H743
- **优势**：成本低、可定制
- **注意**：需要自行设计硬件、烧录器

## 自研板推荐芯片

### 主控芯片
- **STM32F407**：性价比高，资源充足
- **STM32F427**：更多外设，适合复杂系统
- **STM32H743**：高性能，适合 RT-Thread 等
- **STM32H723**：辽宁科技大学 COD 战队选择

## 常用调试工具

### 烧录器
- **ST-Link V2**：经典选择，稳定可靠
- **J-Link**：功能强大，支持多种芯片
- **DAP-Link**：开源方案，成本低

### 逻辑分析仪
- **USBee AX Pro**：入门级选择
- **Saleae Logic 8**：8 通道，常用
- **LA1010/LA2010**：性价比高

### USB-CAN 适配器
- **USB-CAN II**：周立功，工业级
- **CANalyst-II**：性价比高，开源驱动
- **USB-CAN 分析仪**：淘宝常见品牌

## 视觉常用硬件

### 小屏幕
- **0.96寸 OLED**：I2C 接口，显示状态
- **2.4寸 TFT LCD**：SPI 接口，显示图像
- **树莓派官方屏**：7寸，DPI 接口

### 视觉相机
- **USB 摄像头**：罗技 C920/C930，通用选择
- **工业相机**：大恒/大华，支持触发
- **深度相机**：Intel Realsense，ROS 友好

### 视觉调试方案

#### 有线方案
- **HDMI/DP 直连**：稳定、延迟低
- **USB 视频采集卡**：可录屏、可调试

#### 无线方案
- **图传模块**：大疆图传、轻量图传
- **HDMI 延长器**：有线可靠距离
- **无线投屏器**：商用方案，需配对

#### 远程调试方案
- **HDMI/DP 欺骗器**：模拟显示器，无屏启动
- **ToDesk/TeamViewer/向日葵**：远程桌面
- **SSH + ROS RViz**：有线/无线局域网

## 机械常用电气

### 继电器
- **SRD-05VDC-SL-C**：5V 继电器
- **G5V-1**：信号继电器，低功耗

### 开关
- **微动开关**：限位检测、行程开关
- **按钮开关**：手动控制

### 连接器
- **XT60/XT30**：电源接口
- **PH2.0/XH2.54**：信号接口
- **JST GH**：紧凑可靠

## 视觉工业相机/USB 摄像头

### USB 摄像头推荐
- **罗技 C920e**：1080p，稳定驱动
- **罗技 C930e**：更广角度，适合哨兵
- **电商通用 USB 摄像头**：低成本，需验证

### 工业相机推荐
- **大恒水星系列**：MER-132/230
- **大华华睿**：A3138MG000
- **Basler**：acA系列，高性能

### 选型要点
- **帧率**：30fps 最低，60fps 推荐
- **分辨率**：640x480 或 1280x720
- **接口**：USB3.0 > USB2.0
- **镜头**：焦距 6mm/8mm，匹配视场

## 导航雷达选型

### 激光雷达
- **Livox Mid360**：深圳北理莫斯科大学使用
  - 测距范围：0.1-40m
  - 扫描方式：固态，无旋转
  - ROS 驱动：官方支持
- **Sick TIM561/571**：经典选择
- **思岚 RPLIDAR A1**：低成本入门
- **LD19 LD06**：低预算方案

### 选型要点
- **测距范围**：战场需求 10-30m
- **刷新率**：10Hz 最低，20Hz 推荐
- **ROS 支持**：必须有成熟驱动
- **重量体积**：适合哨兵或步兵
