# RoboMaster 硬件系统

## 硬件系统分层架构

RoboMaster 机器人采用典型的三级分层架构：

### 控制层
- **主控制器**：负责决策、协调、指令下发
- **核心组件**：STM32 开发板、外设接口

### 传感层
- **传感器**：IMU、编码器、激光雷达、相机
- **功能**：环境感知、状态反馈

### 执行层
- **执行器**：电机、舵机、发射机构
- **功能**：具体动作执行

## 控制器选型

### C 型开发板
赛事专用高性能控制主板：
- **主控芯片**：STM32F4 系列 32 位 ARM Cortex-M4
- **内置功能**：陀螺仪、加速度计集成 IMU
- **通信接口**：2 路 CAN 总线、多个 USART 串口
- **工作电压**：5V - 24V DC
- **典型应用**：云台控制、多传感器融合、复杂运动控制

### A 型开发板
同样基于 STM32F4 平台，专为底盘控制优化：
- **特点**：丰富 PWM 输出，适合多电机驱动
- **完整 CAN 接口**：支持分布式控制架构

## 电机系统详解

### M3508 减速直流电机
赛事最主流底盘驱动电机：
- **额定电压**：DC 24V
- **峰值扭矩**：3.7 N·m
- **额定扭矩**：2.5 N·m
- **空载转速**：480 rpm
- **减速比**：19:1
- **配套电调**：C620

### M2006 P36 直流无刷减速电机
精密控制场景优化：
- **额定电压**：DC 24V
- **峰值扭矩**：0.75 N·m
- **额定扭矩**：0.5 N·m
- **减速比**：36:1
- **配套电调**：C610

### GM6020 高速电机
云台专用一体化电机：
- **额定电压**：DC 24V
- **空载转速**：320 rpm
- **额定扭矩**：1.2 N·m
- **额定电流**：1.62A
- **内置电调**：无需外部驱动

### 电机扭矩排序
M3508 > GM6020 > M2006

## 电源系统设计

### 电池规格
- **类型**：LiPo 锂电池
- **常见配置**：3S (11.1V)、4S (14.8V)
- **容量范围**：2200mAh - 6000mAh
- **放电倍率**：25C - 50C

### 电源分配架构
- **主电源总线**：直接采用电池电压（4S 为 24V）
- **DC-DC 转换模块**：产生 12V、5V、3.3V 多级电压

### 电源设计原则
- **冗余设计**：关键系统双路供电
- **独立保险丝**：各子系统单独配置
- **电磁兼容**：完整 EMC 设计
