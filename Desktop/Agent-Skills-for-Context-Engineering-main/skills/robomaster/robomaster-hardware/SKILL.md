# RoboMaster 硬件系统

## 一、硬件分层架构

```
┌───────────────────────────────────────────┐
│               控制层 (Controller)         │
│  A 型/C 型 STM32 开发板 · 自研板         │
│  决策 · 协调 · 指令下发                  │
└──────────────────┬────────────────────────┘
                   │ CAN / SPI / I2C / UART
┌──────────────────▼────────────────────────┐
│               传感层 (Sensors)            │
│  IMU(BMI088/ICM42688) · 编码器           │
│  LiDAR · 相机 · 裁判系统模块             │
└──────────────────┬────────────────────────┘
                   │ 反馈数据
┌──────────────────▼────────────────────────┐
│               执行层 (Actuators)          │
│  M3508 + C620 · M2006 + C610             │
│  GM6020 · 发射机构 · 舵机               │
└───────────────────────────────────────────┘
```

## 二、控制器（开发板）详解

### C 型开发板

| 参数 | 规格 |
|------|------|
| **主控芯片** | STM32F407IGH6 (ARM Cortex-M4, 168MHz) |
| **Flash/RAM** | 1MB / 192KB |
| **内置传感器** | BMI088 六轴 IMU (陀螺仪+加速度计) |
| **IST8310 磁力计** | 三轴地磁传感器 |
| **CAN 总线** | 2 路 (CAN1: PB8/PB9, CAN2: PB12/PB13) |
| **UART** | 6 路 USART + 2 路 UART |
| **PWM** | 8 路高级定时器输出 |
| **SPI/I2C** | 3 路 SPI, 3 路 I2C |
| **输入电压** | 5V-24V DC (宽电压) |
| **典型应用** | 云台控制、传感器融合、视觉通信 |

**引脚资源映射**：
```
CAN1_TX: PD1 或 PB8     CAN1_RX: PD0 或 PB9
CAN2_TX: PB12 或 PB5    CAN2_RX: PB13 或 PB6
UART1:  PA9(TX)/PA10(RX)    UART3: PB10/PB11
UART6: PC6(TX)/PC7(RX)      SPI1:  PA5-PA7
I2C1:   PB6(SCL)/PB7(SDA)   PWM:   TIM1_CH1-4, TIM8_CH1-4
```

### A 型开发板

| 参数 | 规格 |
|------|------|
| **主控芯片** | STM32F427IIH6 (ARM Cortex-M4, 180MHz) |
| **Flash/RAM** | 2MB / 256KB |
| **CAN 总线** | 2 路 |
| **PWM** | 丰富，适合多电机 |
| **特色** | 底盘运动学解算优化 |
| **典型应用** | 底盘控制、电机驱动 |

## 五、电源系统设计

### LiPo 电池规格

| 参数 | 3S 方案 | 4S 方案(推荐) |
|------|---------|---------------|
| **标称电压** | 11.1V | 14.8V |
| **满电电压** | 12.6V | 16.8V |
| **容量** | 2200-3300mAh | 3300-6000mAh |
| **放电倍率** | 25C+ | 45C+ |
| **适用场景** | 小型机器人 | 标准步兵/英雄 |

### 电源树设计

```
电池 4S (14.8V ~ 16.8V)
├── 主供电 24V (升压模块) ──► M3508/C620, C型板
├── DC-DC 12V ──► 图传模块, 工业相机
├── DC-DC 5V  ──► 逻辑电路, 传感器
│   └── LDO 3.3V ──► STM32 内核, OLED
└── 独立 5V(隔离) ──► 裁判系统模块
```

### DC-DC 选型参考

| 电压 | 推荐模块 | 电流 | 用途 |
|------|---------|------|------|
| 5V | LM2596 / MP1584 | 3A | 通用逻辑 |
| 12V | XL4015 | 5A | 相机/图传 |
| 24V 升压 | XL6019 | 5A | 主功率 |

### 电源设计原则

1. **冗余设计**：关键系统双路供电
2. **独立保险丝**：各子系统单独配置
3. **电磁兼容**：π 型滤波 + 共模扼流圈
4. **电源监控**：ADC 实时监测电池电压

## 六、PCB 与电磁兼容设计

### 布局原则
- 功率区与信号区分离
- CAN 终端电阻放置于总线末端(120Ω)
- IMU 远离大电流走线
- 模拟地与数字地单点连接

### 布线要点
- 功率走线宽度 ≥ 2mm (I≤5A)
- CAN 差分走线(120Ω 阻抗)
- 信号走线避开功率区

### EMC 防护
- π 型 LC 滤波(输入侧)
- TVS 管(电源入口)
- 共模扼流圈(CAN 总线)
