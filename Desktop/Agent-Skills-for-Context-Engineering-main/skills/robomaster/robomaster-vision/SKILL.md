# RoboMaster 视觉系统

## 系统架构

视觉系统采用分层架构设计：

### 图像采集层
- USB 摄像头
- 工业相机
- 深度相机
- 图传模块
- 常见分辨率：640x480、1280x720
- 帧率：30-60 fps

### 图像预处理层
- 图像去噪
- 颜色空间转换（RGB→HSV）
- 感兴趣区域（ROI）提取
- 直方图均衡化
- 边缘检测
- 形态学操作

### 目标识别层
- 装甲板检测
- 能量机关识别
- 敌方机器人分类
- 算法选择：
  - 传统视觉（颜色分割、形状检测）
  - 深度学习（YOLO、CNN）

### 决策输出层
- 目标坐标输出
- 装甲板类型识别
- 威胁等级评估
- 射击决策

## 传统视觉算法

### 装甲板检测流程
1. **颜色空间转换**：RGB → HSV
2. **颜色阈值分割**：蓝色/红色装甲板
3. **形态学操作**：开运算、闭运算
4. **轮廓检测**：寻找目标轮廓
5. **特征匹配**：长宽比、面积判断

### 关键参数
- 颜色阈值：需要根据光照条件调整
- 轮廓面积：过滤过小或过大的轮廓
- 长宽比：装甲板通常为 2:1

## 深度学习方案

### YOLO 系列
- YOLOv5：成熟稳定，资源要求适中
- YOLOv8：最新版本，性能更好
- 优势：端到端检测，实时性好
- 部署平台：NVIDIA Jetson 系列

### 关键点检测
- 装甲板四点模型
- 能量机关五点模型
- 区域赛视觉识别板检测

## 目标跟踪与预测

### 卡尔曼滤波
用于目标运动预测，补偿通信延迟和控制滞后。

关键步骤：
1. 状态预测
2. 测量更新
3. 协方差更新

### 平滑滤波
使用 α 滤波实现目标位置平滑：
```
x = α * measurement_x + (1 - α) * x
y = α * measurement_y + (1 - α) * y
```

## 部署优化

### 边缘推理
- 模型量化：FP32 → FP16 → INT8
- TensorRT 加速
- ONNX 格式转换

### 实时性优化
- ROI 裁剪，只处理感兴趣区域
- 多线程处理，流水线并行
- 跳帧检测，降低帧率
